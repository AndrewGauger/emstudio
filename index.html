EMStudio

Cross Platform FreeEMS tuning application, written in c++ with Qt

Download as .zip Download as .tar.gz View on GitHub

How do I get it?

The easiest way to get EMStudio is to download the source from the repository and compile it yourself. A few packages are needed to compile it. In ubuntu:

$ sudo apt-get update
$ sudo apt-get install libqt4-dev libqt4-opengl-dev qt4-qmake libqwt-dev libqt4-declarative libqjson-dev

Once this has completed (or the equivalant packages on your own distro), you can then clone the repository and build it. It has been mentioned before to run qmake -project, NEVER EVER EVER DO THIS!. This will break things. To checkout and build:

$ git clone https://github.com/malcom2073/emstudio.git
$ cd emstudio
$ qmake
$ make

There are two main branches that are typically available. master, and fredtest. Please run master, as that is where I will accept bugreports from.

Once it has compiled, it will be available to run via emstudio binary in the main directory. There are also precompiled windows installers, that are in no way garunteed to work, since they pull from master for each commit. These are available at http://mikesshop.net/EMStudio. You may need the MSVC++ 2010 runtime to run these.

What can it do?

EMStudio is a work in progress. As such, its feature list is slightly limited as of now. Currently, it can:
* Display datalog data via gauges, or text table
* Load, edit, and write raw hex RAM memory locations or 2D tables
* Show ECU Flags
* Look Awesome
