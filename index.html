EMStudio

Cross Platform FreeEMS tuning application, written in c++ with Qt

Download as .zip Download as .tar.gz View on GitHub

How do I get it?

The easiest way to get EMStudio is to download the source from the repository and compile it yourself:

$ git checkout https://github.com/malcom2073/freetune.git
$ qmake
$ make

Once it has compiled, it will be available to run via emstudio binary in the main directory. There are also precompiled windows installers, that are in no way garunteed to work, since they pull from master for each commit. These are available at http://mikesshop.net/EMStudio

What can it do?

EMStudio is a work in progress. As such, its feature list is slightly limited as of now. Currently, it can:
* Display datalog data via gauges, or text table
* Load, edit, and write raw hex RAM memory locations or 2D tables
* Show ECU Flags
* Look Awesome
